### Shooting script

[0-2s] HOOK
  On-screen: "Solo developers are a bottleneck. Agent teams are the fix."
  Spoken/VO: "Solo developers are the bottleneck. Agent teams are the fix — and they already work."
  Visual: Fast cut from a lone developer staring at a screen to a pipeline dashboard showing three agents labelled Planner / Executor / Tester lighting up. Motion from frame 1.

[2-8s] THE CLAIM
  On-screen: "You don't need a larger team. You need a smarter one."
  Spoken/VO: "You don't need to hire more engineers. You need to stop being the only one executing."
  Visual: Simple diagram: one person icon handing a spec doc to three agent icons who each take a lane and start running.

[8-12s] TICK 1 → 2 (THE PROOF BEGINS)
  On-screen: "Tick 1 — Planning" / "Tick 2 — Building"
  Spoken/VO: "I gave a 3-agent team one product spec. Tick one: they planned every task themselves. Tick two: they wrote the code."
  Visual: Screen-recording: planner agent output scrolling → code file being written by the executor. Fast cuts.

[12-16s] TICK 3 → 4 (PROOF COMPLETE)
  On-screen: "Tick 3 — Testing" / "Tick 4 — Done."
  Spoken/VO: "Tick three: they tested it and caught their own bugs. Tick four: done. I didn't touch a thing between any of them."
  Visual: Test runner output → terminal line "All tasks complete." Brief hold.

[16-22s] THE REFRAME
  On-screen: "This is the Hello World of agent teams."
  Spoken/VO: "Remember running Hello World for the first time — that exact moment when it just worked? This is that moment for autonomous agent teams."
  Visual: Browser opens showing the working to-do app. Brief click-through to confirm it's real.

[22-28s] THE IMPLICATION
  On-screen: "If they can close a full task now — what's next?"
  Spoken/VO: "If a team of agents can take a spec to a working application with zero intervention — what happens when the specs get harder?"
  Visual: App still visible. Slow hold or subtle zoom.

[28-30s] CTA
  On-screen: "Comment: what would YOU give them to build?"
  Spoken/VO: "Drop a comment — what would you give your agent team to build first?"
  Visual: Final frame on the app.

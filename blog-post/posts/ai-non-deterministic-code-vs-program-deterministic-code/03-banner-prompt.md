## Banner Image Prompt

**Core metaphor:** Two mirrored gears — one machined and perfectly toothed, turning in a fixed, rigid loop; the other made of soft, shifting smoke or liquid metal, changing shape slightly each rotation. Same axle, same input motion, visibly different fidelity of output.

**Prompt (universal base):**
A dark, minimal editorial illustration contrasting order and drift. On the left, a precise mechanical gear, perfectly machined, locked into a clean repeating rotation, sharp edges, cold blue light. On the right, a second gear made of shifting smoke or liquid metal, same size and position, but its shape subtly deforms with each turn, warm amber light bleeding at its edges. Both gears connected by a single shared axle in the center, symbolizing one input feeding two different kinds of output. Deep charcoal background, high contrast, sparse negative space, moody editorial lighting, sense of quiet tension between control and unpredictability. Wide banner composition, 16:9.

**Tool modifiers:**
- Midjourney: append `--ar 16:9 --style raw --v 6`
- DALL·E / GPT-4o: add "digital illustration, flat design, no gradients"
- Ideogram: add "flat vector illustration, editorial style, tech blog"

**Do NOT include:** code on screens, company logos, text or titles, photorealism

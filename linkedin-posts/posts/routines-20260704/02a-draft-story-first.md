# Draft A — Story-first

Last week, at 8:30 PM, I wasn't thinking about LinkedIn.

I was winding down. Laptop closed.

But somewhere in the cloud, an agent had already cloned my content repo, scanned my idea backlog, drafted a post, generated images, and opened a pull request for me to review in the morning.

I didn't run it. I didn't trigger it. It had a job to do.

That's the shift Claude Code Routines made for me.

Not "AI helps me write things."

More like: the AI has a schedule, a brief, and a definition of done.

The bottleneck stopped being memory or motivation.

It became: did I write a clear enough prompt?

I run three of these now. Each one clones the repo, does its task, and opens a PR. No babysitting required.

Close the laptop. The work keeps going.

If you're using AI tools manually today, what's the first task you'd set to auto-run?
